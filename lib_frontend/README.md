# Digital Sanctuary — React Frontend

React + TypeScript + Vite + Tailwind CSS implementation of the [Figma design](https://www.figma.com/design/9UsG1fcvmFw92ct0PxPdbw/Untitled?node-id=0-1).

## Screens

| Route | Figma frame |
|-------|-------------|
| `/login` | Student Login |
| `/student` | Student Dashboard |
| `/history` | Borrow History |
| `/scanner` | Book Scanner |
| `/admin` | Admin Dashboard |
| `/lending` | Borrow Requests |
| `/inventory` | Inventory Management |
| `/returns` | Return Station |

Open `/` for a screen index.

## Development

```bash
npm install
npm run dev
```

Images load from Figma MCP asset URLs (valid ~7 days). Re-export from Figma if assets expire.
